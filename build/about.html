Автоматизировання система определения риска первичной резекции кишки при опухолевой толстокишечной непроходимости

Введение

Автоматизировання система определения риска первичной резекции кишки при опухолевой толстокишечной непроходимости будет использоваться для автоматизации процессов прохождения, составления, редактирования анкеты, составления и редактирования справочников с отделениями и должностями, работы с врачами (регистрация врачей, изменение информации о врачах, редактирование списка пациентов врача), работы с пациентами (просмотр информации, просмотр и составление рекомендаций и направлений на лечение, контроль правильности введённых данных, просмотр результатов прохождения анкеты), просмотра статистической информации и вывода информации о пациентах, а также направлений на лечение на печать.

Программа использует базу данных Microsoft SQL Server Management Studio 18. Программная система совместима с Windows 7 и выше. Для пользования программой необходимо предустановить браузер - Google Chrome 86.0.4240.183 (64-битный) и выше или Firefox 83.0 (64-битный) и вышe.

Авторизация, запуск программы

Данная программная система поддерживает три роли пользователей: роль администратора, врача и пациента. При запуске программной системы открывается страница с анкетой, которую можно пройти незарегистрированному пользователю и получить первичную рекомендацию, составленную системой на основе результатов прохождения анкеты. В случае, если обнаружен высокий риск, цвет текста с первичной рекомендацией красный, иначе - зелёный.

При нажатии на элемент верхнего навигационного меню "Войти" открывается страница авторизации, представленная на рисунках. Необходимо ввести логин и пароль в соответствующие поля, после чего пользователь может нажать кнопку "Войти!" или "Назад" для входа в систему или возврата на главную страницу. У пользователя есть возможность посмотреть вписанный пароль или скрыть его, нажав на кнопку "Показать пароль" и "Скрыть пароль" соответственно.

При нажатии на элемент верхнего навигационного меню "Зарегистрироваться" открывается страница регистрации пациента, представленная на рисунке. Необходимо ввести фамилию, имя, отчество, пол, СНИЛС, номер телефона, адрес, полис, дату рождения, email (при наличии), логин и пароль в соответствующие поля, после чего пользователь может нажать кнопку "Зарегистрироваться" или "Назад" для регистрации новой учётной записи пациента или возврата на главную страницу. У пользователя также есть возможность посмотреть вписанный пароль или скрыть его, нажав на кнопку "Показать пароль" и "Скрыть пароль" соответственно.

Если данные при регистрации введены верно, то система уведомляет пользователя об успешной регистрации новой учётной записи. В случае неудачи система выдаёт сообщение о введённых некорректных данных.

Если данные при входе введены корректно и пользователь с такой учетной записью зарегистрирован, то система определяет роль пользователя и отображает соответствующий интерфейс. На рисунках представлены интерфейсы пользователя после успешной авторизации в роли пациента, врача и администратора.

Если данные при входе введены некорректно, то система уведомляет пользователя об ошибке.

Администратор

Основная задача администратора – работа с анкетой. Он может добавлять новый или редактировать уже имеющийся вопрос. Также у него есть возможность удаления вопросов из анкеты.

При работе со справочниками отделений или должностей администратор имеет возможность: добавления, изменения и удаления названий отделений или должностей соответственно.

При работе с врачами он может регистрировать в системе новую или удалять имеющуюся учётную запись врача, редактировать информацию уже имеющегося в системе врача, а также вести список его пациентов.

При работе с пациентами администратор может контролировать правильность введённых данных пациентом, подтверждать его учётную запись или удалять из системы, просматривать сведения о нём.

Работа с анкетой

Для работы с анкетой администратору необходимо выбрать пункт меню "Исходная анкета". После этого на странице отобразится текущий список вопросов анкеты, элементами этого списка являются блоки, которые содержат текстовые поля для ввода текста вопроса, текста ответа, числовое поле напротив каждого ответа для задания веса ответа. Также имеются кнопки для удаления каждого из ответов, кнопка для добавления нового ответа в список ответов и кнопка "Удалить" под текстом вопроса, при нажатии на которую удаляется вопрос из списка. Над списком распоолагаются две кнопки: "Добавить", при нажатии на которую создаётся элемент списка вопросов анкеты, и кнопка для сохранения всех изменений в базе данных.

В случае, если все элементы списка имеют корректное содержание: текстовые поля для вопроса и ответов не пустые, а также сам список ответов не пустой, то система при сохранении выдаёт сообщение об успешной операции. Иначе система не сохраняет список и выдаёт сообщение об ошибке.

Работа со справочниками

Для редактирования справочников отделений и должностей администратор должен выбрать пункт меню "Справочники". Система отображает страницу, на которой содержатся два списка в соответствии со справочниками должностей и отделений. Каждый список состоит из элементов, которые содержат текстовое поле для ввода названия должности или отделения, а также кнопку напротив каждого текстового поля для удаления элемента из списка. Над каждым списком имеются две кнопки, которые отвечают за добавление нового элемента в соответствующий список и сохранение списка в базу данных.

При сохранении, в случае если список содержит пустой или повторяющийся элемент, то администратору выдаётся соответствующее сообщение. Иначе система сохраняет все изменения в базу данных и тоже уведомляет об этом.

Работа с врачами

Для работы с врачами администратору необходимо выбрать пункт меню "Врачи". Система отображает таблицу с информацией о врачах. Каждая строка таблицы состоит из текстовых полей, предназначенных для редактирования фамилии, имени и отчества врача, двух выпадающих списков, элементами которых являются названия должностей и отделений соответственно. Также в строке содержится кнопка "Пациенты" для перехода на страницу с пациентами соответствующего врача и кнопка, предназначенная для удаления учётной записи врача. Сверху над таблицей располагаются кнопки "Добавить", при нажатии на которую система отображает страницу для регистрации новой учётной записи врача и кнопка "Сохранить", при нажатии на которую соответствующие изменения сохраняются в базе данных.

Страница, которая отображается при регистрации новой учётной записи врача, представлена на рисунке. Она содержит текстовые поля для ввода фамилии, имени, отчества, логина и пароля, а также два выпадающих списка для задания отделения и должности врача. Внизу располагаются две кнопки: "Зарегистрировать" и "Назад" для регистрации и возврата на страницу с таблицей врачей соответственно. У администратора также есть возможность посмотреть вписанный пароль или скрыть его, нажав на кнопку "Показать пароль" и "Скрыть пароль" соответственно.

При нажатии на кнопку "Зарегистрировать" система отображает соответствующее сообщение. Если введены корректные данные, то сообщение об успехе, иначе сообщение о некорректном вводе.

При переходе на страницу с пациентами соответствующего врача, система отображает таблицу с информацией о пациентах врача. Каждая строка таблицы состоит из текстовых полей, предназначенных только для просмотра фамилии, имени и отчества, даты рождения, пола, СНИЛСА, телефона, адреса, полиса и email. Также в строке содержится кнопка "Анкеты" для перехода на страницу с пройденными анкетами и кнопка, предназначенная для отвязки пациента от данного врача. Над таблицей располагается кнопка "Добавить" и выпадающий список с подтверждёнными пациентами, которые ещё не привязаны ни к одному врачу. Для того, чтобы привязать врачу нового пациента, необходимо выбрать этого пациента в выпадающем списке и нажать кнопку "Добавить". Также над таблицей находится кнопка, при нажатии на которую соответствующие изменения сохраняются в базу данных. Слева расположена кнопка со стрелкой, которая предназначена для возврата на страницу со списком врачей.

При нажатии на кнопку "Анкеты" система отображает список пройденных пациентом анкет. При нажатии на соответствующий элемент открывается страница с результатом прохождения выбранной анкеты. Каждая страница содержит кнопку со стрелкой для возврата на предыдущую страницу.

Работа с пациентами

Для работы с пациентами администратору необходимо выбрать пункт меню "Пациенты". Система отображает страницу "Пациенты". Если есть неподтверждённые учётные записи пациентов, то поверх страницы появляется модальное окно с таблицей с информацией об этих пациентах (ФИО, дата рождения, пол, СНИЛС, телефон, адрес, полис и email). Напротив каждой записи есть кнопка, которая предназначена для подтверждения, и кнопка для отказа в регистрации учётной записи. При нажатии на кнопку "ОК" модальное окно скрывается и отображается страница, совпадающая со страницей с пациентами отдельного врача, но уже в этой таблице информация о всех подтвержденных пациентах, а также добавляется новая колонка с ФИО врача, к которому привязан пациент.

Врач

Основная функция врача - работа с пациентами, то есть наблюдение за пациентами и ведение их медицинской карты. В медицинскую карту входят как рекомендации пациенту, которые врач может составлять и редактировать, так и направления на его последующее лечение. Направления и информацию о пациенте врач может выводить на печать.

У врача имеется возможность просмотра результатов прохождения анкетирования. Ему доступны варианты ответов, которые выбрал пациент, на соответствующие вопросы.

Также врач в любое время может посмотреть статистику по заболеваемости среди всех пациентов по таким основным признакам, как возраст и пол пациентов.

Работа с пациентами

Для работы с пациентами администратору необходимо выбрать пункт меню "Пациенты". При переходе на страницу с пациентами соответствующего врача, система отображает таблицу с информацией о его пациентах. Каждая строка таблицы состоит из текстовых полей, предназначенных только для просмотра фамилии, имени и отчества, даты рождения, пола, СНИЛСА, телефона, адреса, полиса и email. Также в строке содержатся кнопки: "Рекомендации", "Направления", при нажатии на которые врач попадает на страницу со списком рекомендаций и направлений выбранного пациента соответственно, кнопка "Анкеты" для перехода на страницу с пройденными анкетами и кнопка, предназначенная для вывода информации о пациенте на печать. Слева расположена кнопка со стрелкой, которая предназначена для возврата на главную страницу.

При нажатии на кнопку "Рекомендации", система отображает список рекомендаций выбранного пациента. При наведении курсором на элемент списка, система показывает всплывающую подсказку с текстом рекомендации. Каждый элемент списка содержит кнопку "X" для удаления рекомендации. При нажатии на соответствующий элемент поверх страницы открывается модальное окно с текстовым полем, который уже содержит текст рекомендации, это текстовое поле доступно для редактирования. Под текстовым полем располагаются две кнопки: "Сохранить" для сохранения изменений и "Отмена" для отмены внесенных изменений. Над списком рекомендаций находится кнопка "Добавить", которая также открывает модальное окно с пустым текстовым полем с целью добавления новой рекомендации. Слева расположена кнопка со стрелкой, которая предназначена для возврата на страницу с пациентами.

При нажатии на кнопку "Направления", система отображает список направлений выбранного пациента. При наведении курсором на элемент списка, система показывает всплывающую подсказку с текстом направления. Каждый элемент списка содержит кнопку "X" для удаления направления и кнопку для вывода текста направления на печать. При нажатии на соответствующий элемент поверх страницы открывается модальное окно с текстовым полем, который уже содержит текст направления, это текстовое поле доступно для редактирования. Под текстовым полем располагаются две кнопки: "Сохранить" для сохранения изменений и "Отмена" для отмены внесенных изменений. Над списком направления находится кнопка "Добавить", которая также открывает модальное окно с пустым текстовым полем с целью добавления нового направления. Слева расположена кнопка со стрелкой, которая предназначена для возврата на страницу с пациентами.

Статистика по заболеваемости

Для просмотра текущей статистики среди пациентов о необходимости первичной резекции кишки при опухолевой толстокишечной непроходимости по основным признакам врачу необходимо выбрать пункт меню "Статистика". Слева от диаграммы будут расположены кнопки: "Пол" и "Возраст" для выбора по какому именно признаку формировать диаграмму со статистикой.

Пациент

Основная функция пациента - прохождение анкеты. Для этого он должен ответить на все вопросы и отправить результат. Также пациент имеет возможность просмотра рекомендаций и направлений на лечение, которые выписывает его лечащий врач.

Прохождение анкеты

На главной странице после успешной авторизации в качестве пациента сразу доступна анкета для прохождения анкетирования. Необходимо указать пол и возраст. После чего нажать на кнопку "Далее". В том случае, если что-то не указано, система выдает сообщение об ошибке и не переходит на следующую страницу.

В случае успеха система отображает следующие вопросы анкеты. Под ними расположена кнопка "Отправить", чтобы отправить и сохранить результат прохождения анкетирования. При нажатии на неё, если ответы даны на все вопросы, система отображает страницу с результатом и первичной рекомендацией, снизу располагается кнопка "Пройти ещё раз" для прохождения анкеты повторно. Если же на какой-то вопрос пациент не ответил, то выдаётся сообщение об ошибке и выделяются вопросы, на которые не были даны ответы.

Просмотр рекомендаций и направлений

Для просмотра своих рекомендаций пациент должен выбрать в верхнем меню пункт "Рекомендации". Система отображает список рекомендаций конкретного пациента. При наведении курсором на элемент списка, система показывает всплывающую подсказку с текстом рекомендации. При нажатии на соответствующий элемент поверх страницы открывается модальное окно в целях наглядности с текстовым полем, который уже содержит текст рекомендации. Под текстовым полем располагается кнопка "Назад", которая скрывает модальное окно. Слева расположена кнопка со стрелкой, которая предназначена для возврата на главную страницу.

Для просмотра своих направлений на лечение пациент должен выбрать в верхнем меню пункт "Направления". Система отображает список направлений конкретного пациента. При наведении курсором на элемент списка, система показывает всплывающую подсказку с текстом направления. При нажатии на соответствующий элемент поверх страницы открывается модальное окно в целях наглядности с текстовым полем, который уже содержит текст направления. Под текстовым полем располагается кнопка "Назад", которая скрывает модальное окно. Слева расположена кнопка со стрелкой, которая предназначена для возврата на главную страницу.
